    # 字符串的常用方法

    # 操作方法

    • 增
    • 删
    • 改
    • 查

    # 增

    增不是原有字符串上增，而是生成副本后进行操作

    ES6 ${}拼接

    # concat

    let stringValue = "hello " 
let result = stringValue.concat("world")
console.log(result)//hello world

    # 删

    同理不是删除原有字符串，而是生成副本后进行操作

    • slice()
    • substr()
    • substring()
    let stringValue = "hello world"
console.log (stringValue.slice(3))//"lo world"
console.log (stringValue.substring(3))//"lo world"
console.log (stringValue.substr(3))//"lo world"
console.log (stringValue.slice(3,7))//"lo w"
console.log (stringValue.substring(3,7))//"lo w"
console.log (stringValue.substr(3,7))//"lo worl" 不同的是substr第二个参数是结果长度

    # 改

    同理不是直接改变原有字符串,而是生成副本后进行操作

    • trim() trimLeft() trimRight()
    • repeat()
    • padStart() padEnd()
    • toLowerCase() toUpperCase()

    # trim() trimLeft() trimRight()

    删除前后或者仅后仅前的空格符，中间的空格符忽略

    let stringValue = " hello world "
let trimmedStirngValue = stringValue.trim()

    # repeat()

    接收整数参数表示重复次数

    let stringValue = "na "
let copyRes = stringValue.repeat(10)

    # padEnd() padStart()

    常用于日期格式化补零 复制字符串，并判断长度，若小于指定长度，end则末尾补符号 start则前端补对应符号

    let stringValue = "foo"
console.log(stringValue.padStart(6))//仅长度参数只返回本身
console.log(stringValue.padStart(6,"*"))//"***foo"

    # toLowerCase() toUpperCase()

    字符大小写转化

    # 查

    除了如同使用索引脚标来获取字符串的值，还可以通过以下方法

    • chatAt()
    • indexOf()
    • startWith()
    • includes()

    ndexOf()方法与includes()方法的一个重要区别在于indexOf()并不能判断数组的NaN元素，换句话说，不管数组arr是否有NaN元素，arr.indexOf(NaN)返回值都是-1，

    # chatAt()

    返回索引位置字符，参数是整数参数

    # indexOf()

    从字符串开头查找是否有字符,如果找到就返回找到的第一个位置索引，没有找到就返回-1

    # startWith() includes()

    字符串中搜索子字符串，返回是否存在的布尔值

    # 转换方法

    围绕着数组与字符串的转化

    数组to字符串：

    1. toString 自带的转换api
    2. toLocate String() 把数组转化为在特定语言环境下指定的字符串 比如时间Date
    3. join 数组按指定分隔符连接字符串 默认逗号 可指定空或者空格

    字符串to数组：

    1. split 相当于join的逆操作 第一参数是字符串或者正则表达式 指定分割（删除匹配）位置 第二参数是输出的最大长度
    2. ES6 拓展运算符
