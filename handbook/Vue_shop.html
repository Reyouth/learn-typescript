    Vue脚手架下使用Element-UI实现电商页面

    ZRY 2022/3/1

    # 技术栈简述

    # Vue

    Vue (读音 /vjuː/，类似于 view) 是一套用于构建用户界面的渐进式框架

    # Node.js

    传统的js是运行在浏览器上的，因为浏览器内核分为两个部分:渲染引擎---渲染HTML和CSS。Chrome 使用的 JavaScript 引擎是 V8，性能很优越。

    # 什么是Node.js

    Node.js 是一个基于 Chrome V8 引擎的 JavaScript 运行环境。Node.js 使用了一个事件驱动、非阻塞式 I/O 的模型，使其轻量又高效。

    # Node 工具

    npm（包管理工具）是基于 Node.js的前端项目包管理工具，是项目中对各种程序包的依赖管理

    webpack是基于 Node.js的前端项目部署打包工具（前端工程化）

    Express 是一个保持最小规模的灵活的 Node.js Web 应用程序开发框架，为 Web 和移动应用程序提供一组强大的功能。

    # JSON-SERVER实战

    在实战VUE之前先了解下json-server，顾名思义json服务器，在我们的入门实战中，不想引入太过于复杂的后端环境但是又想体验模块化的前端整个HTTP请求的完整过程，所幸Nodejs提供了json-server工具，方便快速的搭建简易服务器。

    # 安装json-server

    npm i json-server -g

    i 指 install -g 指 global

    # 运行

    创建一个db.json文件，内容如下： { "data": [] }

    然后在该目录下打开命令行窗口，执行

    json-server --watch db.json

    回显如下说明成功

    Loading db.json Done

    Resources http://localhost:3000/data (opens new window)

    Home http://localhost:3000 (opens new window)

    Type s + enter at any time to create a snapshot of the database Watching...`

    # 测试

    http://localhost:3000/data (opens new window) 是接口地址，直接用curl发送POST,DELETE,PATCH,GET请求模拟增删改查，只要测试一个通过了即可。

    包括了增删改查，分别是POST DELETE PATCH GET

    # Vue实战

    # 安装Vue-cli

    vue-cli即vue的脚手架，脚手架指有快速创建模板化项目的工具，此处使用vue2.0版本

    npm install -g vue-cli

    vue3.0 是 @vue/cli

    # 安装webpack

    webpack可以将vue项目打包成静态文件

    npm install webpack -g

    # vue 2.0 + Element UI 项目搭建

    按照脚手架的提示

    初始化一个叫做vue2_elementui的项目，使用webpack模板 vue init webpack-simple <project name>

    进入项目目录 cd <project name>

    构建 npm i

    运行 npm run dev

    npm run 可以在package.json 的scripts 内自定义调试等操作

    # 引入Element UI

    退出当前终端

    安装Element UI npm install element-ui -S

    顺便安装vue-router 和 vue-resource，前者是路由，后者是执行Ajax请求用到的依赖 npm install vue-router vue-resource --S

    安装moment，事件选择组件 npm install moment -S

    -S指当前项目安装 跟全局安装区分

    # Element UI 的按需导入使用

    # 安装babel-plugin-component

    借助 babel-plugin-component，我们可以只引入需要的组件，以达到减小项目体积的目的

    npm install babel-plugin-component -D

    # 更改.babelrc文件
    "plugins": [
    [
      "component",
      {
        "libraryName": "element-ui",
        "styleLibraryName": "theme-chalk"
      }
    ]
  ]
    # 在 src 文件夹中新建我们的 element 文件夹 新建index.js

    部分导入举例如下

    import { Select, Option, OptionGroup, Input, Tree, Dialog, Row, Col } from 'element-ui'
const element = {
  install: function (Vue) {
    Vue.use(Select)
    Vue.use(Option)
    Vue.use(OptionGroup)
    Vue.use(Input)
    Vue.use(Tree)
    Vue.use(Dialog)
    Vue.use(Row)
    Vue.use(Col)
  }
}
export default element
    # 在 main.js 中导入该自定义的element文件

    注意：按需加载跟全局加载只能存在一个
