    如何得到最接近的三数之和

    # 如何得到最接近的三数之和

    # 问题描述

    给定一个包括 n 个整数的数组 nums 和 一个目标值 target。找出 nums 中的三个整数，使得它们的和与 target 最接近。返回这三个数的和。假定每组输入只存在唯一答案。

    # 排序+双指针法

    在排序之后，左小右大，i从0到length-1 遍历，L初始在i右侧，R初始在最右侧

    sum = nums[i]+nums[L]+nums[R]

    function threeSum(arr, target) {
  arr = arr.sort((a, b) => a - b);//js冒泡排序最简写法，若a大于b则b在前a在后
  var res = arr[0] + arr[1] + arr[2];
  for (var i = 0; i < arr.length; i++) {
    var low = i + 1;
    var high = arr.length - 1;
    while (low < high) {
      var sum = arr[i] + arr[low] + arr[high];
      // 将三数之和与target进行比较，如果比target大，则右指针左移，如果比target小，左指针右移
      if (sum > target) {
        high--;
      } else {
        low++;
      }
      // 之后再判断当前的三个数之和与之前的res相比谁的差值更小，差值用绝对值
      if (Math.abs(sum - target) < Math.abs(res - target)) {
        res = sum;
      }
    }
  }
  return res;
}

    i本身代表枝干，决定了小到大的遍历，双指针的和又影响其中的大小，若sum大则移动R指针，若sum小了则移动L指针以此来进行拟合
