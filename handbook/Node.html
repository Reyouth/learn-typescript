    Node.js 一个JS的运行环境

    # Node.js 一个JS的运行环境

    Nodejs 是基于 Chrome 的 V8 引擎开发的一个 C++ 程序，目的是提供一个 JS 的运行环境。 node.js可以辅助前端开发 尽管nodejs不属于前端，但是node.js因为使用的语言是js。所以可以很好的辅助前端开发. node.js繁荣了前端的构建工具

    # npm

    npm 是 Node Package Manager 的缩写，意思是 Node 的包管理系统。Nodejs 现在如日中天，其中 npm (opens new window) 这个功不可没。在这里，我们要实现各种功能几乎都能找到现成的别人写好的包，直接拿了用就好了。

    很多 npm 包都对应一个 Github 项目，但是如果只有代码，那么使用起来还不是特别方便。而当系统上安装好了 Node.js 之后，就会配套安装一个命令，叫做 npm

    npm install moment

    # Express

    Express 是一个简洁而灵活的 node.js Web应用框架使用npm安装。express的出现使得可以用js代码简洁明了地创建web应用。

    npm i express 安装express

    var express = require('express');//使用前需导入并创建实例
var app = express()

    核心特性

    • 中间件响应http请求
    • 路由表响应http请求
    • 向模板传参动态渲染HTML页面

    # 请求响应

    请求响应基于回调函数内实现

    app.get('/', function (req, res) {
   // --
})

    其中req为http请求，里面包含了查询字符串参数，httpheader等信息。

    res为响应数据，可以定义向客户端发送的响应数据

    # 路由

    请求响应定义了请求响应动作，而路由定义特定的脚本去响应

    app.get('/', function (req, res) {
   console.log("主页 GET 请求");
   res.send('Hello GET');
})
 
 
//  POST 请求
app.post('/', function (req, res) {
   console.log("主页 POST 请求");
   res.send('Hello POST');
})

    # 数据库模块的初始化

    进行数据库模块的初始化，数据库模块已经封装好在./modules/databse内，里面包含了初始化initialize函数

    //app.js
var database = require('./modules/database')
database.initialize(app, function(err) {
  if (err) {
    console.error('连接数据库失败失败 %s', err)
  }
})

    database模块主要使用了mysql与orm模块，其中后者是基于node来创建数据模型，实现将声明的模型同步添加到数据库 即关系映射在node的实现。

    //database.js 核心代码

	var opts = {//
		protocol : config.get("protocol"),
		host : config.get("host"),
		database : config.get("database"),
		port : config.get("port"),
		user : config.get("user"),
		password : config.get("password"),
		query : {pool: true,debug: true}
	};


   	app.use(orm.express(opts, {
		define: function (db, models, next) {

			app.db = db;
			global.database = db;

			// 获取映射文件路径
			var modelsPath = path.join(process.cwd(),"/models");
			
			// 读取所有模型文件 模型文件可以理解为数据库存的各种数据的格式 比如权限模型 角色模型
			fs.readdir(modelsPath,function(err, files) {
				// 存放所有的加载模型函数
				var loadModelAsynFns = new Array();
				// console.log("开始加载 ORM 模型层文件 ");
				for (var i = 0; i < files.length; i++) {
					var modelPath = modelsPath + "/" +files[i];
					// console.log("加载模型 %s",modelPath);
					loadModelAsynFns[i] = db.loadAsync(modelPath);
				}
				
				Promise.all(loadModelAsynFns)//等待所有完成或者接收第一个失败 异步等待处理加载模型的完成与否
				.then(function(){//所有完成
					// console.log("ORM 模型加载完成");
					// 挂载模型集合

					for(var modelName in db.models){
						models[modelName] = db.models[modelName];
					}
					app.models = models;
					callback(null);
					next();
				})
				.catch(function(error){//接收失败
					console.error('加载模块出错 error: ' + err);
					callback(error);
					next();
				});
			});
		}
	}));
